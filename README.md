# Pinterest-style Image Grid

Welcome to the Pinterest-style Image Grid website! This project displays a dynamic grid of random images pulled from the [Picsum Photos API](https://picsum.photos). It features an interactive image gallery with the ability to view enlarged images, a back-to-top button, and a loading spinner for better user experience.

## Demo

You can check out the live demo of the site here:  
[https://anasbolm123.github.io/anas-bolm/](https://anasbolm123.github.io/anas-bolm/)

## Features

- **Dynamic Image Grid**: Displays random images in a responsive grid layout.
- **Modal for Enlarged Images**: Click on any image to view it in a larger view in a modal.
- **Back-to-Top Button**: A button that allows users to quickly scroll back to the top of the page.
- **Loading Spinner**: A spinner that appears while images are loading.
- **Responsive Design**: The layout adapts to various screen sizes.

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **[Picsum Photos API](https://picsum.photos)** for fetching random images.
